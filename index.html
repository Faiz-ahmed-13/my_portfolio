GRID-X: AI Race Strategy & Telemetry Analytics

Built comprehensive ML system analyzing historical F1 telemetry data (1950-2024) with Random Forest lap-time predictor, XGBoost ensemble classifier, DQN agent for pit-stop optimization, and transfer learning for circuit recognition. Deployed unified FastAPI backend with Docker containerization.

Random Forest XGBoost DQN ResNet50 FastAPI Docker PyTorch
  • Random Forest lap-time prediction – Achieved RMSE 2.787s (97% accuracy, R² 0.993) using 21 telemetry and environmental features.
  • XGBoost ensemble – Dual‑era classifier with logistic regression stacking, delivering 96.4% win accuracy and 93.2% podium accuracy.
  • Unsupervised driver clustering – KMeans identified 4 distinct driving archetypes (Aggressive, Smooth, Opportunistic, Balanced) from telemetry data.
  • Reinforcement learning pit‑stop optimizer – DQN agent trained in a custom gym environment with tyre degradation, learning optimal 2‑stop strategies for 52‑lap races.
  • Crash risk prediction – XGBoost classifier with ROC‑AUC 0.80 and 74% recall for estimating race incident probability and safety car likelihood.
  • Tire safety assessment – RandomForest model providing real‑time risk scores (0‑100) with SHAP explanations, enabling proactive pit decisions.
  • Computer vision circuit recognition – Fine‑tuned ResNet50 with transfer learning achieving 97.6% accuracy on 24 circuit schematics.
  • SHAP explainability – Integrated for lap‑time and tire safety predictions, giving feature‑level transparency.
  • Unified REST API – FastAPI backend serving all models for real‑time and batch inference, with comprehensive Swagger documentation.

Premier Energies Solar ETL Agent

Built a fully automated 5-node ETL pipeline during Premier Energies internship. Ingests solar panel sales CSV data, applies GST pricing and business rules, deduplicates 200 → 179 records, loads clean data into a cloud Postgres database, and fires a real-time Slack alert — all in ~5 seconds with zero manual steps. Open-sourced as company data is under NDA.

Agentic AI n8n Neon Postgres Slack Webhooks JavaScript ETL Pipeline Cloud DB
  • Agentic pipeline: 5 automated n8n nodes running end-to-end without any manual intervention
  • Business rules engine: GST calc (₹25/W × 1.18), segment standardization, missing efficiency fill at 18.5%
  • Deduplication: composite key (panel_type + sale_date) reduced 200 rows to 179 clean records
  • Cloud DB: idempotent TRUNCATE + INSERT into Neon Postgres — safe to re-run at any time
  • Real-time alert: Slack Block Kit webhook fires on every successful run with revenue summary
  • ₹39.06 Crore total revenue processed across Monocrystalline & Polycrystalline panel types
